# 📊 Ecommerce Sales Dashboard

## 🔹 Project Overview

This project focuses on analyzing ecommerce sales data using **Python, SQL, and Power BI** to generate business insights and build an interactive dashboard.

## 🔹 Tools Used

* Python (Pandas, NumPy)
* SQL
* Power BI
* Jupyter Notebook
* GitHub

## 🔹 Project Workflow

1. Data Cleaning using Python
2. Data Analysis using SQL
3. Data Visualization using Power BI
4. Dashboard Creation

## 🔹 Key Insights

* Region-wise sales performance
* Product-wise sales analysis
* Monthly sales trend
* Category-wise performance
* Profit analysis
